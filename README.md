## 🚀 Overview
The goal of this portfolio is to provide a clean, high-performance, and responsive interface to present my analytical projects, professional experience, and technical toolkit. It reflects my focus on Business Insights, SEO Analytics, and Data Automation.

## 🛠️ Built With
Frontend: HTML5, Modern CSS3 (CSS Variables, Flexbox, Grid)

Animations: CSS Keyframes & JavaScript Intersection Observer API (Scroll Reveal)

Typography: Google Fonts (Instrument Serif, Manrope, JetBrains Mono)

Deployment: GitHub Pages

Dynamic Projects Section: Showcases key work like the Pinterest Analytics Tool, including technical stack tags and project status indicators.

Interactive UI: Custom-built cursor effects, smooth scrolling, and an email contact modal.

Responsive Design: Fully optimized for mobile, tablet, and desktop viewing.

Scroll Reveal System: Sophisticated entrance animations for improved storytelling and user engagement.
